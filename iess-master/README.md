# iess

#### 介绍
面经分享系统

#### 软件架构
软件架构说明
基础登录模块（t_user）（登录，注册，记住我，退出登陆），+手机验证

个人中心模块（修改资料，修改密码，全局异常处理，非法拦截模块，收藏模块）拓展功能阿里云oss

面经分享（包括，交流讨论(讨论帖+一个评论区)
圈子（每日打卡模块，吐槽模块，许愿池，摸鱼圈等）(讨论帖+一个评论区)

面经公司模块

统计报表模块，

系统公告模块（即首页展示功能）

系统管理模块（权限管理，班长管理，组长管理，组员权限）
